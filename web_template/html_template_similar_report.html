Thank you for using YTPO!

This report was generated automatically using a CSV file. It includes all videos with similar titles. This report can be very useful if you need to identify and manage duplicate videos in your playlists.

Disclaimer! This tool only compares similarity in video titles. It may generate inaccurate results and does not compare actual video content. Videos with completely different titles will not be listed as duplicates. You can adjust the similarity tolerance in config.ini, located in the program's root directory.
